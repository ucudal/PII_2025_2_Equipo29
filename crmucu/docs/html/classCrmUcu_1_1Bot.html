Detailed Description

Clase principal del bot de Discord. Gestiona la conexión con Discord y delega el manejo de comandos a CommandHandler. Aplica SRP: solo se encarga de la conexión y lifecycle del bot. Aplica Controller: actúa como punto de entrada para eventos de Discord.
