Detailed Description

Manejador central de comandos del bot. Aplica Command Pattern: cada comando es una clase que implementa ICommand. Aplica Controller: recibe mensajes de Discord y delega a comandos específicos. Aplica SRP: solo se encarga de parsear mensajes y ejecutar comandos. Aplica OCP: agregar nuevos comandos solo requiere registrarlos en el diccionario.
