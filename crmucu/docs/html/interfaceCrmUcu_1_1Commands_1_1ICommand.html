Detailed Description

Interfaz que define el contrato para todos los comandos del sistema. Aplica Command Pattern: encapsula acciones como objetos ejecutables. Aplica ISP (Interface Segregation): interfaz pequeña y específica. Aplica OCP: nuevos comandos se agregan sin modificar código existente. Permite polimorfismo: todos los comandos se ejecutan uniformemente.
